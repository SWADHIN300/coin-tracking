# Coin Tracking Project

## Project Description
This project tracks cryptocurrency prices and provides insights into the market. It allows users to monitor trends, set price alerts, and view historical data.

## Features
- Track multiple cryptocurrencies
- Real-time price updates
- Price alerts via email
- Historical price charts

## Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- Database: MongoDB

## Installation Instructions
1. Clone the repository: `git clone https://github.com/SWADHIN300/coin-tracking.git`
2. Navigate to the project directory: `cd coin-tracking`
3. Install dependencies: `npm install`

## Build Commands
Run the following command to build the project:
```bash
npm run build
```

## Deployment Details
To deploy the project:
1. Ensure that your environment variables are set.
2. Use a service like Heroku or Vercel to deploy the app.
